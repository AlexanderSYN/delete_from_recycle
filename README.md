# delete files from recycle auto
# while program only for russia (EN version is lower)

# RU
# программа создана для очистки корзины после заданое время (Only Windows 10/11)

# все команды

help - для помощи
```
help
```

info - для информации о программы
```
info
```
add (расширение) - добавить расширения (add all - удаляет сразу все файлы из корзины)
```
add (extension)
```

del (расширение) - удалить расширение (del all - удаляет сразу все расширения, которые вы добавили)
```
del (extension)
```

time (время в секундах) - изменить время удаления файлов
```
time (sec)
```

ext / extensions - вывод всех расширений файлов, которые вы удаляете
```
ext
```
```
extensions
```

clrec - очистить корзину
```
clrec
```

cls / clear - очистить консоль
```
cls
```
```
clear
```

ex / exit (или Ctrl + C) - для выхода из программы
```
ex
```
```
exit
```
